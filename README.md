🏋️‍♂️ GymQuest

A Gamified Gym Desktop App (Java PBL Project)

📌 Description

GymQuest is a Java-based desktop application that transforms traditional workouts into an engaging game-like experience. Users can earn XP, level up, complete challenges, and track their fitness journey in a fun and interactive way.


---

🚀 Features

🎮 Gamification of workouts (XP, Levels, Rewards)

🏆 Daily & Weekly Challenges

📊 Progress Tracking (reps, sets, workouts)

👤 User Profiles

💾 Data persistence using File Handling

🖥️ Interactive GUI using Java Swing

🔔 Motivational alerts & feedback



---

🛠️ Tech Stack

Language: Java

GUI Framework: Swing (Javax Swing)

Core Concepts Used:

OOP (Encapsulation, Inheritance, Polymorphism)

File Handling

Event Handling

Exception Handling

Collections Framework

🎯 How It Works

1. User creates a profile


2. Logs workouts (reps, sets, exercises)


3. Earns XP based on activity


4. Levels up after reaching XP thresholds


5. Completes challenges to unlock rewards
